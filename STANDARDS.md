# Standards

## Local workspace (macOS)
- Code: `~/projects/active/<repo>`
- Archive/duplicates: `~/projects/archive/`
- Assets (audio/video/exports): `~/projects/assets/` (do not commit)

## Git
- Default branch: `main`
- Small commits, clear messages
- Always add `.gitignore` early (node_modules, dist/build, env files, audio exports)

## Naming
- repos: kebab-case
- docs: UPPERCASE.md for meta (README, ROADMAP, STANDARDS, REPOS)
